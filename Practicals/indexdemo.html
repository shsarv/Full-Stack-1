About Course

Front-end web development is everything involved in programming the user interface of a web application. Typically it refers to the Hyper text Markup Language(HTML), Cascading Style Sheets(CSS) and Java Script portion of web site production as opposed to the database or server side programming. It encompasses everything from buliding a simple page of html text to creating complex,responsive HTML5 websites designed to be accessed via various different browsers , devices and screen sizes.

Tools used for front-end development

Here are several tools and platforms (wordpress, magento etc..) available that can be used to develop the front end of a website, and understanding which tools are best fit for specific tasks marks the difference between developing a hacked site and a well designed, scalable site.
